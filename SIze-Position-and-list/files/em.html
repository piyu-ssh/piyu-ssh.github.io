In this webpage, I will show you about units in CSS, more specifically about em unit!

Short me samaj loo em unit ka main use hota ha aapne parent element ka font size, jaise me dhikata hu-

Ye mene ek div banaya container ke name se and me isme ye paragraph likh rha hu, thik,

ye text jo me likh rha hu vo child-container me likha hu,

aab hoga kya-

                
.container {
    font-size: 18px;
    color: cornflowerblue;
}
.child-container {
    font-size: 1.1em;
    color: darkred;
}

Jaise isme .container ka size mene "18px" set kia, and I know ki iska child element ko mujhe thoda sa bada show karna ha, to mene .child-container ka size 1.1 em de diya, jo 18*1.1 = 19.8 px ho gya ha, (aap ye cheez inspect karke khud se bhi verify kar sakte hain!)
