Basically, REM ka matlab related to root element hota ha, jaise em tha vaise REM ha, bass ye root element ke corrosponding size set karta ha!

jaise mene ye div banay, iski size HTML (jo ki root element ha, usse badhegi, ya ghategi, jaise me rem ka use karunga ...)

CSS code-

            
html {
  font-family: "Lora", serif;
  /* By default, the font size of HTML (root element) is 16px*/
}

.div1 {
    font-size:  1.5rem;
}
